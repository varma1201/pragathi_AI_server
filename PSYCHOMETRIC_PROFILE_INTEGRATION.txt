This document describes the psychometric assessment and user profile system
that provides personalized idea validation based on entrepreneur characteristics.

================================================================================
  SYSTEM OVERVIEW
================================================================================

The system consists of 3 main components:

1. **Psychometric Evaluator** (psychometric_evaluator.py)
   - Generates 10-20 dynamic assessment questions using GPT-4.1
   - Evaluates responses across 10 dimensions
   - Provides comprehensive personality analysis

2. **User Profile Manager** (user_profile_manager.py)
   - Creates entrepreneur profiles from psychometric data
   - Stores profiles in MongoDB
   - Generates personalized validation contexts

3. **Integration Layer** (psychometric_endpoints.py, app_v3.py)
   - API endpoints for assessments and profiles
   - Automatic profile creation after evaluation
   - Profile-aware idea validation

================================================================================
  WORKFLOW
================================================================================

Step 1: User Takes Psychometric Assessment
-------------------------------------------
POST /api/psychometric/generate
- Generates 10-20 questions dynamically
- Questions assess 10 entrepreneurial dimensions
- Takes ~25 seconds with GPT-4.1

Step 2: User Submits Responses
-------------------------------
POST /api/psychometric/evaluate
- Evaluates all responses
- Calculates dimension scores (0-10)
- Generates personality insights
- **Automatically creates user profile**
- Takes ~11 seconds with GPT-4.1

Step 3: Profile is Created & Stored
------------------------------------
- Profile includes:
  ✓ Entrepreneurial fit score
  ✓ Top strengths
  ✓ Development areas
  ✓ Personality summary
  ✓ Validation focus areas
  ✓ Risk tolerance level

Step 4: Idea Validation Uses Profile
-------------------------------------
POST /api/validate-idea
- System retrieves user profile
- Adds personalized insights to validation:
  ✓ User fit score
  ✓ Strengths to leverage
  ✓ Areas needing focus
- Validation history added to profile

================================================================================
  10 PSYCHOMETRIC DIMENSIONS
================================================================================

1. Leadership & Vision
   - Ability to lead teams and set direction
   
2. Risk Tolerance
   - Comfort with uncertainty and calculated risks
   
3. Resilience & Adaptability  
   - Recovery from setbacks and adapting to change
   
4. Innovation & Creativity
   - Creative thinking and novel solutions
   
5. Decision Making
   - Quality and speed of judgment under pressure
   
6. Emotional Intelligence
   - Self-awareness and interpersonal effectiveness
   
7. Persistence & Grit
   - Determination to pursue long-term goals
   
8. Strategic Thinking
   - Analyzing complex situations and planning ahead
   
9. Communication Skills
   - Clarity and effectiveness in conveying ideas
   
10. Problem Solving
    - Analytical and creative approach to challenges

================================================================================
  API ENDPOINTS
================================================================================

Psychometric Assessment
-----------------------
POST   /api/psychometric/generate
       Body: {"num_questions": 10-20, "user_id": "user123"}
       Returns: Question set with options and scoring profiles

POST   /api/psychometric/evaluate
       Body: {
         "user_id": "user123",
         "user_name": "John Doe",
         "questions_data": {...},
         "responses": {"q1": "A", "q2": "B", ...}
       }
       Returns: Complete evaluation + creates profile

User Profiles
-------------
GET    /api/profile/<user_id>
       Returns: Complete user profile

GET    /api/profile/<user_id>/validation-context
       Returns: Personalized context for validation

GET    /api/psychometric/evaluations/<user_id>
       Returns: All evaluations for user

GET    /api/psychometric/evaluation/<evaluation_id>
       Returns: Specific evaluation details

- Address their weak areas proactively

================================================================================
  DATABASE COLLECTIONS
================================================================================

1. psychometric_assessments
   - Stores generated question sets
   - Links to user and evaluation

2. psychometric_evaluations
   - Complete evaluation results
   - Linked to user profile

3. user_profiles
   - Entrepreneur profiles
   - Validation history
   - Updated after each assessment

4. validation_reports
   - Enhanced with profile insights
   - Links back to user profile
